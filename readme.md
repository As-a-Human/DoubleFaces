## 两面人

每个人都有选择自己生活方式的权利，我完全尊重那些选择外国护照或者绿卡的人。
但是有的人一边让自己和家人逃离这个体制，一边又为这个体制摇旗呐喊。总想着在这个体制中喝几口血，对在这个制度中受苦受难的人视而不见，着实恶心。

1. 董卿

> 你们要爱国，要爱五星红旗
2019年9月1日，董卿央视主持『开学第一课』时，她对孩子们说到。

同时，董卿把孩子生在美国。

央视一姐向我们生动的诠释了：爱国是工作，赴美是生活。
